About Node.IQ

01

Node.IQ AB is a new startup based in Eskilstuna, Sweden. We're building edge AI hardware for teams who need inference running close to where the data is made, not shipped off to a data center first.

Our devices are built to be integratable by default — designed to sit alongside the sensors, controllers, and systems already on the floor, rather than asking anyone to rip them out.

What we build

02

Compact inference modules

Small form-factor boards that run models locally, so decisions happen where the data is captured — no round trip to the cloud.

Standard interfaces

Common connectors and protocols so a device slots into an existing line or system without a custom integration project.

Low-power design

Built to run continuously on modest power budgets, for sites without a server room or a data center nearby.

Connected & V2X-ready

Built-in connectivity and V2X support let devices talk to vehicles, infrastructure, and each other — not just report back to a server.

Over-the-air deployment

Ship firmware and model updates remotely, so fleets stay current without a truck roll or a technician on site.
